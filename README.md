Начальные требования:
Рабочий ноутбук/ПК,
Стабильное подключение к интернету,
Установленный Docker Desktop,
Git.
Не заняты порты 8000 и 5432
Установка:
Шаг 1. Клонировать проект к себе командой в cmd, представлен на Рисунке 1:
git clone https://github.com/ZarikZa/kursadeploy.git
Шаг 2. Перейти в основую папку проекта, представлено на Рисунке 2:
cd kursadeploy\hhproject
Шаг 3. Запустить Docker Desktop и убедиться, что он отображается в системной трее, а затем запустить в терминале проект командой, представлено на Рисунке 2:
docker-compose up -d –build
Шаг 4. Настройка БД и запуск скрипта создания суперпользователя, следует выполнить миграции:
docker-compose exec web python manage.py migrate
Далее прописать эту команду:
docker-compose exec web python manage.py shell
После появления надписей об успешной первичной настройке БД выйти из shell прописав exit()
Шаг 5. Доступ к приложению будет выполняться по ссылке из любого браузера:
http://localhost:8000
Админ-панель находится по адресу:
http://localhost:8000/admin
Остановка проекта выполняется в терминале командой:
docker-compose down
